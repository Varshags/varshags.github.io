Projects

List of recent projects that i am proud of

An Automated Coin Recognition System Using GLCM - The aim of the project is to build a coin recognition system using Grey Level Cooccurrence Matrix(GLCM). A camera is used to capture the images of the coins. GLCM features are extracted from the captured image and a feature vector is generated. This feature vector is fed as input to an Artificial Neural Network which is trained to recognize the coins.
An Automated Coin Recognition System using Circlet Transform - The main aim of the project was to build a coin recognition system to recognize Indian denomination coins using Circlet transform. The image of the coin was captured using a camera and Circlet transform was applied on these images to identify the circles. From these circles the GLCM features were extracted and a feature vector was generated which was used to train an Artificial Neural Network to recognize the coins.
